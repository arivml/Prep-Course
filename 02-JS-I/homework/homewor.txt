* Variables
    Es la manera que tenemos de darle valor de algo y almacenarlo para luego utilizarlo

* Strings
    Son bloque de textos que siempre debemos escribilos entre comillas, pueden ser comillas simples (' ') o comillas dobles (" ")

* Funciones (argumentos, `return`)
    Son bloques de construcción muy importantes javascrip, es basicamente un conjunto de instrucciones que realiza una tarea, o calcula un valor.
    Para que su uso sea correcto se debe definir en algun lugar desde el que desees llamarla.

    En las funciones los Argumentos son las variables que se necesitan para poder dar el resultado. La funciòn indicara cuantos argumentos necesita y el nombre de cada uno.

    Return es la manera de finalizar la ejecuciòn de la funcion, de esta manera especifica un valor para ser devuelto a quien llama a la función.

* Declaraciones `if`
    Se utiliza para ejecutar una sentencia si la condición es verdadera. En dado caso que sea falsa, se podria ejecutar otra sentencia, como seguir avanzando.

* Valores booleanos (`true`, `false`)
    Es un concepto similar a el codigo binario ya que estos utilizan un código que es logica booleana. Es decir solo tienen dos opciones verdadero o falso. En javascri se utiliza mucho para saber si algo es verdadero o no. Son representados con las palabras "true" o "false"
